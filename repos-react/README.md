# Desafio 5 (Parte 2)

Mentoria de React JS - Programa Dev for Tech (Turma 01) com [@gamaacademy](https://github.com/gamaacademy):

- Sistema de consulta de repositórios do GitHub utilizando React JS.

## 🚀 Enunciado do desafio proposto

1. Utilize a API do GitHub para criar uma lista com todos os repositórios da sua conta

2. Crie um campo no qual o usuário pode filtrar a lista de repositórios através do nome

## ☕ Executar projeto

1. Faça o clone do projeto: `git clone git@github.com:Dzin/repos-react.git`

2. Acesse a pasta do projeto via terminal e execute os seguintes comandos:

- Download das dependências: `yarn install`

- Executar projeto: `yarn dev`
